Custom lead statuses let you define your own stages for tracking leads through your sales process. Each brand has its own set of custom statuses alongside the system defaults.

### Default vs. Custom Statuses

Every brand starts with a set of default statuses that cannot be edited or deleted. Custom statuses you create are brand-specific and only appear for your brand.

![Sales Pipeline navigation showing the leads table with status column](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-nav-pipeline.png)
You can tell the difference because custom statuses display an **⋯** (three dots) menu with **Edit** and **Delete** options, while default statuses do not.

![Change Status dropdown showing custom statuses with three dots menu and default statuses without](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-default-vs-custom.png)
### Changing a Lead's Status

You can change a lead's status from two places:

**From the leads table:**

1. Navigate to **Sales → Pipeline**.
2. Find the lead in the table and click the **Status** column for that row.
3. The **Change Status** dropdown opens, showing all available statuses with a search bar at the top.
4. Select a status to apply it immediately.

![Clicking the Status column in the leads table to open the Change Status dropdown](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-change-from-table.png)
**From the lead detail panel:**

1. Click on a lead to open their detail panel.
2. Click the **Status** badge below the lead's name.
3. Select a new status from the dropdown.

![Clicking the Status badge in the lead detail panel to open the Change Status dropdown](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-change-from-detail.png)
### Creating a New Status

1. Open the **Change Status** dropdown on any lead (see above).
2. Scroll to the bottom and click **+ Create New Status**.

![Bottom of the Change Status dropdown showing the Create New Status option](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-create-trigger.png)
3. Enter a name for the status.
4. Click the icon on the left to open the icon picker. Browse icons by category or use the search bar, then pick a color using the color picker below the icon grid.
5. Click **Create New Status** to save.

![Create New Status form with name field, icon picker, and color picker](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-create-form.png)
The new status is immediately available for all leads within your brand.

### Editing a Custom Status

1. Open the **Change Status** dropdown on any lead.
2. Hover over the custom status you want to edit and click the **⋯** (three dots) menu.
3. Select **Edit**.
4. Update the name, icon, or color.
5. Save your changes.

![Three dots menu on a custom status showing the Edit option](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-edit-menu.png)
Changes apply to all leads that currently use that status.

### Deleting a Custom Status

1. Open the **Change Status** dropdown on any lead.
2. Hover over the custom status you want to remove and click the **⋯** (three dots) menu.
3. Select **Delete**.

![Three dots menu on a custom status showing the Delete option](https://raw.githubusercontent.com/maxmaeser/helpdesk-screenshots/master/how-to-create-custom-statuses/statuses-delete-menu.png)
Deleting a status removes it from the picker. Leads that were using the deleted status show as having no status until you assign a new one.
