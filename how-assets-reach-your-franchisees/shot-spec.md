# Shot Spec: How Assets Reach Your Franchisees

| # | Shot | UI state to set up | Cursor | Filename | Width |
|---|---|---|---|---|---|
| 1 | Library sidebar with Published to Franchisees filter visible | Library > List view. Set sidebar to show "Published to Franchisees" filter row (globe icon) and at least one collection with the blue "Published" globe pill (e.g., Grand Opening Kit, Logos & Brand Marks). Ensure internal sidebar is fully visible on the left. | none | `library-published-filter.png` | ~840 CSS |
| 2a | Edit Collection modal — Publish toggle OFF | Library > right-click a collection > Edit Collection. Modal is open, focused on the "Publish to all franchisees" row. Toggle is in the OFF position. Helper text reads "Only visible to explicitly shared organisations." | arrow on toggle | `edit-collection-toggle-off.png` | ~840 CSS |
| 2b | Edit Collection modal — Publish toggle ON | Same modal, same collection, but toggle is in the ON position. Helper text reads "Visible to all franchisee organisations." | arrow on toggle | `edit-collection-toggle-on.png` | ~840 CSS |
| 3 | Bulk action bar with Publish to Franchisees option | Library > List view. Select 2–3 asset rows via checkboxes. Bulk action bar appears at the top showing "Publish to Franchisees" or "Publish N to Franchisees" action button. | arrow on Publish button | `library-bulk-publish.png` | ~840 CSS |
| 4 | Location Assets tab with entity library selected | Operations > Locations > open a location > Assets tab. Three-column layout visible. Select an entity library under "Entity Libraries" (e.g., Aspen United). Right column shows a visibility caption: "Visible to franchisees of [Entity Name] and brand agents" clearly legible. | none | `location-assets-tab.png` | ~840 CSS |
| 5 | Location library empty state | Operations > Locations > open a location > Assets tab. Select an entity library with no collections yet. Empty state displays: "+ Add collection" button (bottom of column) and message "No collections in this library yet. Link an existing collection or create one to group these files." | arrow on + Add collection button | `location-assets-empty.png` | ~840 CSS |
| 6 | Yellow orphaned library warning banner | Operations > Locations > open a location > Assets tab. Select an entity library marked "Not linked" or one whose entity has been unlinked. Yellow warning banner is visible at the top of the Assets column: "Franchisees can't see these documents, only brand agents can." | none | `location-assets-orphaned-banner.png` | ~840 CSS |
